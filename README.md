## ğŸ“ How to Use

### For Interview Prep:

1. **Assess your level:** Start with Essential 50 flashcards
2. **Identify weak areas:** Focus on topics you struggle with
3. **Follow a study path:** Choose 4-week, 12-week, or daily practice
4. **Practice coding:** Solve 2-3 problems daily
5. **Review flashcards:** Use spaced repetition (Day 1, 3, 7, 14, 30)

### For Learning:

1. **Read Q&A sequentially:** Go through one topic at a time
2. **Run code examples:** Copy-paste and experiment
3. **Build projects:** Apply concepts in real applications
4. **Teach others:** Best way to solidify knowledge

### For Last-Minute Review:

1. **Pre-Interview 30 flashcards** (15 minutes before interview)
2. **Skim Essential 50** (highlights of all topics)
3. **Review company-specific questions** (if interviewing at FAANG)
