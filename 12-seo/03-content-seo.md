### Question
How does semantic HTML improve SEO?

### Answer

**Semantic Elements Help Search Engines:**
- Understand content structure
- Identify main content vs navigation
- Determine content hierarchy

```html
<!-- ‚úÖ Semantic (Better for SEO) -->
<header>
  <h1>Main Heading</h1>
  <nav>
    <a href="/">Home</a>
  </nav>
</header>

<main>
  <article>
    <h2>Article Title</h2>
    <p>Content...</p>
  </article>
</main>

<footer>
  <p>&copy; 2025</p>
</footer>

<!-- ‚ùå Non-Semantic (Poor for SEO) -->
<div class="header">
  <div class="title">Main Heading</div>
  <div class="nav">
    <a href="/">Home</a>
  </div>
</div>
```

**Heading Hierarchy:**
```html
<h1>Page Title</h1>  <!-- Only one H1 -->
  <h2>Section 1</h2>
    <h3>Subsection 1.1</h3>
    <h3>Subsection 1.2</h3>
  <h2>Section 2</h2>
```
